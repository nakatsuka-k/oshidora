# デザイン適用 進捗

更新日: 2026-01-28

このファイルは `docs/デザイン` の画像（画面デザイン）を基準に、モバイル実装へ反映した状況を記録します。

## 対応方針

- `docs/デザイン/*.png` のファイル名 → `apps/mobile/screens/*Screen.tsx` を対応付け
- 1画面ずつ「確認→適用→動作/型チェック→記録」を繰り返す
- 既に反映済みっぽい画面（トップ/スプラッシュ/規約など）は大きく弄らず、差分がある場合のみ調整
