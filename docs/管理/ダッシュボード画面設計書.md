# 管理画面 ダッシュボード画面 設計書

---

## 画面名

* 画面ID：CMS-DASHBOARD-001
* 画面名：ダッシュボード
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：
  推しドラ全体の状況を一目で把握し、各管理機能への起点となる画面

---

## 画面位置づけ

* **管理画面ログイン後の初期表示画面**
* 全管理機能へのハブ（入口）

---

## 画面遷移

### 遷移元

* 管理画面ログイン完了画面
* 管理画面内 各画面から「ダッシュボード」メニュー選択

### 遷移先

* 作品管理一覧
* 動画管理一覧
* コメント管理一覧
* キャスト・スタッフ管理
* コイン管理（購入／換金）
* ユーザー管理
* お問い合わせ管理
* 各種設定画面

---

## 画面構成（全体）

```
[ ヘッダー ]
 ├ ロゴ
 ├ 管理者名
 └ ログアウト

[ サイドメニュー ]
 ├ ダッシュボード
 ├ 作品管理
 ├ 動画管理
 ├ キャスト・スタッフ管理
 ├ コメント管理
 ├ コイン管理
 ├ ユーザー管理
 ├ お問い合わせ管理
 └ 設定

[ メインコンテンツ ]
 ├ サマリー情報（KPI）
 ├ 最近のアクティビティ
 ├ 管理ショートカット
```

---

## メインコンテンツ詳細

---

### ① サマリー情報（KPIエリア）

**目的**
サービスの状態を即座に把握する

#### 表示項目例

| 項目       | 内容         |
| -------- | ---------- |
| 総ユーザー数   | 登録ユーザー数    |
| 本日の新規登録  | 当日登録ユーザー   |
| 公開中作品数   | 現在公開中の作品数  |
| 公開中動画数   | 動画総数       |
| 本日の再生回数  | 全動画合算      |
| 本日のコイン消費 | 投げ銭・購入消費合計 |

※ 数値はすべてリンク可能（該当一覧へ遷移）

---

### ② 最近のアクティビティ

**目的**
直近の運営対応が必要な情報を確認

#### 表示内容（例）

* 新規投稿されたコメント（最新5件）
* コメント通報（未対応）
* キャスト・スタッフ新規登録
* コイン換金申請（未処理）
* お問い合わせ（未対応）

#### 表示仕様

* 各行クリックで詳細画面へ遷移
* 未対応件数は強調表示

---

### ③ 管理ショートカット

**目的**
よく使う管理機能へ即アクセス

#### ショートカット例

* 作品を追加
* 動画を追加
* コメント承認
* キャスト登録
* コイン換金処理
* お問い合わせ確認

---

## 画面項目定義（サマリー）

| 項目名    | 種別 | 内容       |
| ------ | -- | -------- |
| 総ユーザー数 | 数値 | 累計登録ユーザー |
| 本日新規   | 数値 | 当日登録     |
| 作品数    | 数値 | 公開作品     |
| 動画数    | 数値 | 公開動画     |
| 再生回数   | 数値 | 当日合計     |
| コイン消費  | 数値 | 当日合計     |

---

## 操作仕様

* ダッシュボードは **閲覧専用**
* 編集・登録操作は各管理画面で実施
* 数値クリック → 対応する管理一覧へ遷移

---

## エラーハンドリング

* データ取得失敗時

  * 「情報を取得できませんでした」表示
  * 画面自体は表示継続
* 権限不足

  * ログイン画面へリダイレクト

---

## 非機能要件

* 初期表示速度：2秒以内
* データはキャッシュ利用可（リアルタイム必須でない）
* モバイル対応不要（管理画面はPC前提）

---

## セキュリティ

* 認証必須
* URL直叩き時も認証チェック
* 戻る操作による情報表示防止
