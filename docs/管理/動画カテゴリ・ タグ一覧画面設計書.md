# 管理画面 動画カテゴリ・タグ一覧画面 設計書

---

## 画面名

* 画面ID：CMS-VIDEO-META-LIST-001
* 画面名：動画カテゴリ・タグ一覧
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：
  動画に紐づく **カテゴリ・タグの作成／編集／管理** を行う

---

## 画面位置づけ

* 管理画面 サイドメニュー「動画管理」配下
* 動画管理におけるマスタ管理画面

---

## 画面遷移

### 遷移元

* ダッシュボード
* 管理画面 動画一覧画面

### 遷移先

* カテゴリ新規作成
* カテゴリ編集
* タグ新規作成
* タグ編集

※ 一覧画面内でモーダル編集も可（実装方針により）

---

## 画面構成（全体）

```
[ ヘッダー ]
 ├ ロゴ
 ├ 管理者名
 └ ログアウト

[ サイドメニュー ]
 ├ ダッシュボード
 ├ 作品管理
 ├ 動画管理
 │   ├ 動画一覧
 │   └ カテゴリ・タグ管理（選択中）
 ├ キャスト・スタッフ管理
 ├ コメント管理
 ├ コイン管理
 └ 設定

[ メインコンテンツ ]
 ├ タブ切り替え（カテゴリ / タグ）
 ├ 一覧テーブル
 └ ページネーション
```

---

## タブ切り替え

* **カテゴリ一覧**
* **タグ一覧**

※ 切り替え時に画面遷移せず、同一画面内で表示切替

---

## カテゴリ一覧（タブ）

### 表示項目

| 項目     | 内容             |
| ------ | -------------- |
| カテゴリID | システムID         |
| カテゴリ名  | 表示用名称          |
| 説明     | 管理用説明          |
| 紐づき動画数 | 対象動画件数         |
| 表示順    | 並び順            |
| 公開状態   | 有効 / 無効        |
| 登録日    | 作成日時           |
| 操作     | 編集 / 有効切替 / 削除 |

---

### 操作

* **新規カテゴリ作成**
* **編集**
* **有効 / 無効切替**
* **削除**

  * 紐づき動画がある場合は削除不可 or 確認表示

---

## タグ一覧（タブ）

### 表示項目

| 項目    | 内容      |
| ----- | ------- |
| タグID  | システムID  |
| タグ名   | 表示用名称   |
| 使用動画数 | 使用件数    |
| 登録日   | 作成日時    |
| 操作    | 編集 / 削除 |

---

### 操作

* **新規タグ作成**
* **編集**
* **削除**

  * 使用中タグは警告表示

---

## 新規作成／編集（共通仕様）

### カテゴリ

| 項目    | 種別       | 備考      |
| ----- | -------- | ------- |
| カテゴリ名 | テキスト     | 必須      |
| 説明    | テキスト     | 任意      |
| 表示順   | 数値       | 小さい順に表示 |
| 有効フラグ | ON / OFF | 初期ON    |

---

### タグ

| 項目  | 種別   | 備考    |
| --- | ---- | ----- |
| タグ名 | テキスト | 必須・一意 |
| 備考  | テキスト | 管理用   |

---

## ページネーション

* 1ページあたり：20件
* 前へ / 次へ
* ページ番号指定

---

## エラーハンドリング

* 重複名称エラー
  →「同じ名称が既に存在します」
* 削除不可エラー
  →「使用中のため削除できません」
* 通信エラー
  → トースト通知で表示

---

## 権限制御

| 操作 | 管理者 | 運営 |
| -- | --- | -- |
| 閲覧 | ○   | ○  |
| 作成 | ○   | ○  |
| 編集 | ○   | ○  |
| 削除 | ○   | ×  |

---

## 非機能要件

* 初期表示：2秒以内
* 非同期通信
* 大量タグでもレスポンス低下しない設計

---

## 備考・仕様補足

* カテゴリは **動画1件につき1カテゴリ**
* タグは **複数付与可能**
* カテゴリ／タグ変更時、既存動画への影響は即時反映
* フロント側動画一覧・検索条件に連動
