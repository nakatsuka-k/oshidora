# 管理画面 動画詳細・編集 画面設計書

---

## 画面名

* 画面ID：CMS_00025
* 画面名：動画詳細・編集
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：動画のプレビュー、説明文編集、シリーズ紐付け、公開状態の制御等を行う

---

## 仕様上の要点（全体機能仕様書より）

* プレビュー
* 説明文編集
* 同シリーズエピソード登録
* 公開 / 非公開制御
* 字幕あり / なしの MP4 を登録
* 再公開時の再承認要否は要検討

---

## 画面遷移

### 遷移元

* 動画一覧（CMS_00024）
* 動画ランキング（CMS_00011）
* おすすめ/ピックアップ設定（CMS_00026/00027/00037/00038）からの詳細導線（任意）

### 遷移先

* （任意）動画プレビュー（同画面内）

---

## 画面構成（全体）

```
[ ヘッダー ]（共通）
[ サイドメニュー ]（共通）

[ メイン ]
 ├ 画面タイトル + 動画基本情報（ID/タイトル/状態）
 ├ プレビュー
 ├ メタ情報編集フォーム
 ├ シリーズ/エピソード設定
 ├ ファイル管理（字幕あり/なし）
 └ 操作（保存/公開/非公開/戻る）
```

---

## 編集項目（例）

実際の項目は既存の動画管理仕様に合わせる。

| 項目 | 種別 | 内容 |
| --- | --- | --- |
| タイトル | テキスト | 編集可否は要件次第 |
| 説明文 | テキストエリア | 編集対象 |
| 公開状態 | トグル/セレクト | 公開/非公開 |
| 出演者 | 参照/選択 | 既存データから選択 |
| カテゴリ | セレクト |  |
| タグ | 複数選択 |  |
