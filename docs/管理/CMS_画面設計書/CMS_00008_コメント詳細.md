# 管理画面 コメント詳細 画面設計書

---

## 画面名

* 画面ID：CMS_00008
* 画面名：コメント詳細
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：コメント内容の確認と承認/否認（および否認理由通知）を行う

---

## 画面遷移

### 遷移元

* 未承認コメント一覧（CMS_00007）

### 遷移先

* 未承認コメント一覧（操作完了後に戻る）

---

## 画面構成（最小）

```
[ ヘッダー ]（共通）
[ サイドメニュー ]（共通）

[ メイン ]
 ├ コメント情報（参照）
 ├ 操作（承認/否認）
 └ 否認理由（否認時のみ）
```

---

## 表示項目（例）

* コメントID
* 投稿日時
* 対象（動画/演者）
* 対象ID/タイトル
* 投稿者
* 本文（全文）

---

## 操作仕様

* 承認：承認実行 → 一覧へ戻る
* 否認：否認理由コメント（必須）→ 否認実行 → 投稿者へ通知（通知方式は運用要検討）
