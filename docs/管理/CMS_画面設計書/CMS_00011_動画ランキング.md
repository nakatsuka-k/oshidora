# 管理画面 動画ランキング 画面設計書

---

## 画面名

* 画面ID：CMS_00011
* 画面名：動画ランキング
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：動画の指標（再生数/評価数/コメント数/コイン数/お気に入り数）をランキングで可視化し、運用判断に利用する

---

## 画面位置づけ

* ダッシュボード（CMS_00002）からの導線
* サイドメニュー「ランキング」配下の起点

---

## 画面遷移

### 遷移元

* ダッシュボード（CMS_00002）
* サイドメニュー（ランキング）

### 遷移先

* 動画詳細・編集（CMS_00025）

---

## 画面構成（全体）

```
[ ヘッダー ]（共通）
[ サイドメニュー ]（共通）

[ メイン ]
 ├ 画面タイトル
 ├ 集計条件（期間/指標/並び順）
 ├ ランキングテーブル
 └ ページネーション
```

---

## 集計条件

仕様書記載より、以下を最低限とする（詳細なソートロジックは要検討）。

* 期間：日付 From-To（任意）
* 指標：再生数 / 評価数 / コメント数 / コイン数 / お気に入り数
* 並び順：降順（デフォルト）/ 昇順

---

## ランキングテーブル

| 列名 | 内容 | 備考 |
| --- | --- | --- |
| 順位 | 1〜 | 同率時の扱いは要検討 |
| 動画ID | システムID | コピー可（任意） |
| タイトル | 動画タイトル | 1行省略表示 |
| 指標値 | 選択中指標の値 | 数値 |
| 公開状態 | 公開/非公開 など | 表示できる範囲で |
| 操作 | 詳細 | CMS_00025へ |

---

## 操作仕様

* 条件変更 → 再検索
* 一覧行クリック（または「詳細」押下）→ CMS_00025
* 取得失敗：画面上部にエラーバナー（例：`ランキングを取得できませんでした`）

---

## 備考

* ソートロジック（集計対象期間、無料/有料、重複排除など）は運用要件により決定する
