# 管理画面 俳優アカウント詳細 画面設計書

---

## 画面名

* 画面ID：CMS_00010
* 画面名：俳優アカウント詳細
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：俳優アカウントの申請内容確認と、承認/非承認（コメント送信）を行う

---

## 画面構成（最小）

```
[ ヘッダー ]（共通）
[ サイドメニュー ]（共通）

[ メイン ]
 ├ 申請者情報（参照）
 ├ 提出情報（参照）
 └ 操作（承認/非承認）
```

---

## 操作仕様

* 承認：承認実行 → 一覧へ戻る
* 非承認：非承認コメント（必須）→ 実行 → 申請者へ送信（方式は運用要検討）
