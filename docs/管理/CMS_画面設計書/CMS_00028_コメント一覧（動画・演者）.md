## 画面名

* 画面ID：CMS_00028
* 画面名：コメント一覧（動画 / 演者）
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：公開済みコメントを対象（動画/演者）で一覧化し、調査・削除等の管理導線を提供する
