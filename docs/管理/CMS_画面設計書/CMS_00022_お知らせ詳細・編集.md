# 管理画面 お知らせ詳細・編集 画面設計書

---

## 画面名

* 画面ID：CMS_00022
* 画面名：お知らせ詳細・編集
* 対象：管理者 / 運営スタッフ
* 種別：Web（管理画面）
* 目的：既存のお知らせ内容を確認し、編集・配信設定（プッシュ通知等）を行う

---

## 画面遷移

### 遷移元

* お知らせ一覧（CMS_00021）

### 遷移先

* （任意）プレビュー表示

---

## 画面構成

```
[ ヘッダー ]（共通）
[ サイドメニュー ]（共通）

[ メイン ]
 ├ 画面タイトル
 ├ 基本情報フォーム
 ├ 配信設定
 ├ プッシュ通知送信設定
 └ 操作（保存/削除/戻る）
```

---

## 基本情報フォーム（例）

| 項目 | 種別 | 内容 |
| --- | --- | --- |
| 配信日時 | 日時 | 予約配信の場合に使用（即時の場合は現在時刻） |
| 件名 | テキスト | 一覧表示のタイトル |
| 本文 | テキストエリア | リッチテキスト有無は要検討 |
| タグ | 複数 | 仕様書に記載あり |
| 登録者 | 表示 | 自動設定（編集不可） |

---

## プッシュ通知送信設定

仕様書記載：プッシュ通知送信設定。

| 項目 | 種別 | 内容 |
| --- | --- | --- |
| 送信する | チェック | ON の場合、配信に合わせて通知送信 |
| 通知タイトル | テキスト | 省略時の扱いは要検討 |
| 通知本文 | テキスト | 省略時の扱いは要検討 |

---

## 操作仕様

* 保存：入力値を検証して更新
* 削除：確認ダイアログ後に削除（論理削除を推奨）
* 取得失敗：`お知らせを取得できませんでした`
* 更新失敗：`お知らせを更新できませんでした`
