## 備考

* 基本仕様は「おすすめ動画設定（CMS_00026）」と同等で、掲載枠の意味が異なる
* ローカルで `http://localhost:8082/pickup` を開く場合、`?api=` 未指定だと **本番API Base（https://api.oshidra.com）** を参照する（表示・保存ともに本番へ反映される）
	* ローカルAPI（wrangler dev）に向けたい場合は `?api=http://localhost:8787` を付与するか、管理サイトの **API Base Override**（/dev）で上書きする
