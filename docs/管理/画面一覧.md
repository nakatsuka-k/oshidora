## 2. ホーム・承認系

* 配信希望日

* 否認理由コメント
* 提出者への通知

### CMS_00007 未承認コメント一覧

* コメント依頼日
* 対象動画
* 提出者

### CMS_00008 コメント詳細

* コメント承認／否認
* 否認時通知

### CMS_00009 未承認俳優アカウント一覧

### CMS_00010 俳優アカウント詳細

* 承認／非承認
* 否認コメント送信
