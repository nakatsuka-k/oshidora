# 推しドラ：全画面まとめ（APP-00001〜APP-00065）

## 共通UI（全画面共通）

### 画面幅（ブラウザ表示時）

* **原則**：ブラウザ表示（Web）では、全画面のコンテンツ領域を中央寄せとし、最大幅を **768px** に制限する（768pxを超えて横に広げない）
* **例外**：トップ（`#/home`）は最大幅を **768px** に制限する
* **備考**：スマホ実機では端末幅に合わせて表示されるため、この制限は主にWeb表示時のレイアウト統一を目的とする

### 背景色（全画面共通・厳守）

* **原則（アプリ側）**：全画面のベース背景色は **`#181815`** で統一する（`#/phone` を含む）
* **実装ルール**：画面の最外周コンテナ（`ScreenContainer` / `SafeAreaView` / ルート`View`）は **`THEME.bg`**（`apps/mobile/components/theme.ts`）を使用し、画面背景色をハードコードしない
* **注意**：カード/モーダル等の「面」や強調表現は別色でよいが、アプリ全体の下地（背面）は常に `#181815` を維持する
